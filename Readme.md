#Checktrend

##概要
* チェックトレンドのiOS(swift)版です。
* [blog](http://blog.livedoor.jp/tattyamm/archives/5077201.html)
* 動作させるには、各ライブラリの導入が必要です。
* 使用ライブラリ
    * Alamofire
    * SVProgressHUD
    * SwifyJSON
    * GoogleAnalytics
    * AdMob
